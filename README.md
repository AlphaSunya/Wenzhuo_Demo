## 关于本作

仙门没落子弟温灼入职黄泉任鬼使一职。

血月中元夜，阴阳两界交汇之时，她只身穿行于人鬼之间——处置恶鬼，护送迷路的魂灵，顺手收拾不长眼的山匪。

然而阴阳秩序之下，每一个鬼魂都有自己说不完的故事。
